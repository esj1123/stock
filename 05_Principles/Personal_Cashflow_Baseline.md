# 개인 고정 현금흐름 기준표

## 운용 메모

- 이 파일은 자동 import 대상 cashflow note가 아니다.
- 증권계좌 이체액은 투자 판단이 아니라 월별 가용 원금 기준으로만 사용한다.
- 변동 지출은 고정 반복 항목으로 확정된 경우에만 추가한다.
- 실제 입출금 이벤트와 broker 원장은 `31_Cashflows/`와 import 산출물에서 별도로 관리한다.
